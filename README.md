
# Stochastic models for materials Science 
### Project 1: Statistical Inference and Distribution Analysis
This project utilizes parametric and non-parametric statistical methods to evaluate environmental data, medical drug efficacy, and stochastic event timing.

# Task 1: Comparative Statistical Studies
Analyzed cadmium contamination levels at two river locations using parallel box plots to visualize distribution spreads.

Performed Shapiro Wilk tests to confirm data followed a normal distribution before applying parametric tests.

Executed F tests to compare variances and one sided t tests to determine if the first location had significantly higher contamination than the second.

Evaluated drug efficacy on patient body temperatures using paired sample logic and scatter plots to visualize relationships.

Estimated the Pearson correlation coefficient and tested its significance to confirm the strength of the relationship between measurements.

# Task 2: Stochastic Modeling and Mean Estimation
Modeled the frequency of coal mine disasters using histograms and Kernel Density Estimation to identify the shape of the underlying distribution.

Calculated ninety five percent confidence intervals for the mean time intervals between disasters to quantify the expected frequency of events.

### Project 2: Advanced Regression and Signal Processing
This project explores complex linear and non-linear modeling techniques to understand material properties and analyze spectroscopic signals.

# Task 1: Material Property Prediction
Developed and compared three different linear regression models using the linear model command to identify the strongest relationship between wood stiffness, elasticity, and density.

Performed residual analysis to check for model abnormalities and identified influential data points using Cooks Distance.

Re-ran regressions after removing influential outliers to ensure the model met standard Gaussian assumptions.

Investigated the impact of interaction terms between explanatory variables using Analysis of Variance to verify if the model was significantly improved.

Validated new material samples by calculating ninety five percent prediction intervals to determine if new data points fit the regressed model.

# Task 2: Non-linear Signal Analysis
Implemented non linear regression to fit a Lorentz model curve to noisy spectroscopic signal data.

Determined optimal parameters for signal curves by selecting appropriate initial values and visualizing the estimated signal against raw data points.

Conducted hypothesis testing to examine if the maximum signal amplitude significantly exceeded defined critical values.
